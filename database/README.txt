
Instructions

Place the contents of this folder in a secure location (e.g. "C:\Program Files\WebCTRL").
Run "install.vbs" to install and start this application as a Windows service.
Use "services.msc" to manually stop or start the service.
The application will generate data files in "%HOMEDRIVE%\WebCTRL Centralizer".
Changes to the configuration file "%HOMEDRIVE%\WebCTRL Centralizer\config.txt" can be made only while the service is stopped.
Use "uninstall.vbs" to uninstall this service and to delete all data files.
